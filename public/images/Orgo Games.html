Organic Chemistry Studying Games

Name that Alkane!
  • Name the unbranched alkane according to the IUPAC system.
  • You got 20 seconds.
  • You have three strikes before the game ends.
  • A strike is assigned if an incorrect answer is given, or the timer hits 0.
Name that Alkene!
  • Name the unbranched alkene according to the IUPAC system.
  • You got 30 seconds.
  • You have three strikes before the game ends.
  • Only use the e,z notation for naming (Cahn–Ingold–Prelog priority rules).
  • If the double bond is at Carbon #1, no 'e-' or 'z-' is placed before the name.
  • The number of the carbon at which the double bond is located is placed after the prefix, but before the suffix.
  • Don't forget dashes!
Newman Projections
  • Rotate the back carbon so that the molecule is in its lowest energy conformtion.
